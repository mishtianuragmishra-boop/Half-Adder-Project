# Half-Adder-Project
Half-Adder Project is a part of the CoddyTech Verilog HDL course that is being documented here,showing how Verilog HDL can be used to demonstrate the Sum and Carry in a Half-Adder using AND and XOR gate 

A Half-Adder uses two inputs say A and B and provides their Sum and Carry where Sum uses the XOR gate and Carry uses the AND gate
A Half-Adder is a combinational logic that adds 2 single-bit binary inputs and provides their Sum and Carry
-Sum(S)
-Carry(C) 
as outputs
## BOOLEAN EXPRESSIONS
Sum=A ⊕ B 
Carry = A & B

## TRUTH TABLE
A  B    S  C
0  0    0  0
0  1    1  0
1  0    1  0
